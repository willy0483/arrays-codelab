Værdier i en array kan tilgås ved hjælp af et indexnummer, hvor det første element har indexnummer 0, det næste har indexnummer 1 og så videre.
For eksempel: 

                        
                        let firstFruit = myFruits[0];
                        console.log(firstFruit); // Output: "æble"

Opgaver

Array oprettelse

opgave 1

Du skal, i filen arrays.js, oprette en array i en constant variabel der hedder myArray. denne array skal indeholde følgende string elementer:

  • 'element1'
  • 'element2'
  • 'element3'
  • 'element4'
  • 'element5'

Array tilgang

opgave 2

Du skal, i filen arrays.js, skrive console.logs, der giver følgende resultat i din console:

  • element3
  • element5
  • element1
  • element4
  • element2

Rækkefølgen er vigtig!!!
